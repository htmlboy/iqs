Basic selectors

The tag selector is its own name, without any additional characters.

The class selector is written with a dot followed by the name of the chosen class. Classes cannot have spaces, special characters (punctuation marks, ñ, ç, `, ´, ^…), nor start with a numeral.

We can create styles that affect a class but only if a specific tag is applied. To achieve this, we write the name of the tag and then the class selector, without spaces.

The space between two selectors indicates that the second is descended from the first. For example, paragraphs within the example class.

We can continue nesting descendant selectors, separating each one by a space.

We can give styles by referring to the id attribute instead of the class. If this is the case, we will use the hash (#) instead of the dot. Whenever we can, it is preferable to use classes, since the id selector is very specific and difficult to override.

The more specific a selector, the more important it is, and this can alter the cascade.

The * selector affects all HTML tags.
