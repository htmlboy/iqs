Color y opacidad

.example{
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

Hay varias formas de asignar un color: palabra clave, hexadecimal, rgb()…

Podemos manipular la opacidad o transparencia a través de la propiedad opacity o el canal alfa de rgba(). 

.example{
  color: hsl(302, 100%, 50%);
}

El sistema hsl() es otra forma de declarar color que puede resultar más intuitiva. 

.example{
  color: hsla(302, 100%, 50%, 0.5);
}

El sistema hsla() funciona igual que hsl(), pero con el canal alfa.

Puedes probar un editor de colores online en hslpicker.com.
