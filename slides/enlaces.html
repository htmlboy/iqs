Enlaces

<a>

La etiqueta <a> se utiliza para crear enlaces a otras páginas web. 

<a href="pagina.html">Ejemplo de texto</a>

El atributo href sirve para indicar la ruta para llegar a la página web que queremos enlazar. 

<a href="http://google.com">Ejemplo de texto</a>

Si la página que enlazamos es externa, tendremos que poner la dirección completa, incluyendo http://. 

<a href="http://google.com" target="_blank">Ejemplo de texto</a>

Si queremos que se cargue en una nueva pestaña, usaremos el atributo target="_blank".

<h3 id="contacto">Ejemplo de texto</h3>

Podemos enlazar a secciones concretas de una página que tengan el atributo id. Los id son identificadores únicos, y no hay dos con el mismo valor en una página.
Nos referiremos a los id con una almohadilla o hash: # 

<a href="mailto:javier@swwweet.com">Envíame un mail</a>

Podemos enlazar a un mail con el prefijo mailto:, y esto abrirá el programa de correo electrónico que haya instalado por defecto en el ordenador.
