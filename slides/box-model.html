All HTML elements are box-shaped. This box has a width and a height.

These boxes have content, space between the content and the border (padding) and space between the border and other boxes (margin).

The width and height properties are used to set the width and height. If we don't set a width, block-type tags tend to be 100% width, and line-type tags depend on their content. If we don't set a height, the box gets the height of the content. 

.example{
  min-width: 200px;
  max-width: 90%;
  min-height: 200px;
  max-height: 1200px;
}

Apart from setting a width or a height, we can also give minimum or maximum widths or heights.

We can give a different value of margin on each side of the box. 

.example{
  margin: 20px;
}

Or the same margin for all 4 sides. 

.example{
  margin: 20px 10px 30px 15px;
}

The margin property is an abbreviated property, which supports up to 4 values separated by spaces.

The padding property follows rules similar to margin. 

*{
  box-sizing: border-box;
}

To prevent padding and border from changing the total width that we have marked with the width property, we can change the box model at the beginning of our CSS, for all the elements.
