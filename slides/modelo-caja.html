Todos los elementos HTML tienen forma de caja. Esta caja tiene una anchura (width), y una altura (height).

Estas cajas tienen contenido, espacio entre el contenido y el borde (padding) y espacio o margen entre el borde y otras cajas (margin).

Las propiedades width y height sirven para fijar la anchura y la altura. Si no fijamos una anchura, las etiquetas de tipo bloque tienden a ocupar el 100%, y los de línea dependen de su contenido. Si no fijamos una altura, la caja adopta la altura del contenido. 

.example{
  min-width: 200px;
  max-width: 90%;
  min-height: 200px;
  max-height: 1200px;
}

Además de fijar un width o un height, también podemos dar anchuras o alturas mínimas o máximas.

Podemos dar un valor de margin diferente a cada lado de la caja. 

.example{
  margin: 20px;
}

O el mismo margen para los 4 lados. 

.example{
  margin: 20px 10px 30px 15px;
}

La propiedad margin es una propiedad abreviada, que admite hasta 4 valores separados por espacios.

El padding sigue unas reglas similares al margin. 

*{
  box-sizing: border-box;
}

Para evitar que el padding y el border alteren la anchura total que hemos marcado con la propiedad width, podemos cambiar el modelo de caja al principio de nuestra CSS, para todos los elementos.
