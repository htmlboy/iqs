Subject goals

  • Learn about the web design industry
  • Plan and design a web site
  • Learn how to write HTML & CSS code
  • Create a responsive web site
  • Publish a web site

Classes

  • During the class, theory will be combined with small exercises and examples.
  • All the sessions will require a computer.
  • Part of each session will be dedicated to a practical activity (design or code something).
  • These practical activities might be finished as homework.
  • Each practical activity will be delivered during the next session and affects qualifications.

Qualification

  • Practical activities in class: 30%
  • Mid-course assignments: 30%
  • Final assignment: 40%

There is no final exam and assignments will be turned in online.

Please

  • Please don't skip classes. They're very practical and it's easy to get lost.
  • Please be on time.
  • Please ask questions if you get lost.
