Fondos

.example {
  background-color: #dd6a92;
}

El color de fondo de un elemento en HTML se declara con background-color. 

.example {
  background-image: url(ruta/imagen.jpg);
}

La propiedad background-image permite asignar una imagen como fondo. 

.example {
  background-image: url(ruta/imagen1.jpg),
                    url(ruta/imagen2.jpg);
}

Podríamos declarar varias imágenes de fondo, separadas por comas.

La propiedad background-repeat nos permite indicar si la imagen de fondo se repite y cómo. La propiedad background-position nos permite alinear la imagen respecto al fondo. 

.example {
  background: hotpink url(imagen.jpg) no-repeat right top;
}

La propiedad background permite asignar los valores de las propiedades de fondo seguidas, separadas por espacios.

La propiedad background-size permite controlar el tamaño al que aparecen las imágenes de fondo.

Con la propiedad background-image también podemos declarar un degradado. En las funciones linear-gradient() o radial-gradient() indicaremos ángulo o posición y colores de corte.
