Introducción a JavaScript

JavaScript es un lenguaje de programación

Programamos comportamientos: ˝cuando se hace click aquí, ocurre algo˝.

Objetivos

  • acceder al contenido de una web
  • modificar el contenido de una web
  • programar reglas
  • reaccionar a eventos

Ejemplo

˝Si yo hago click…˝

˝en un elemento que tenga la clase 'elemento'…˝

˝…quita o ponle la clase 'activo' ˝

Para usar JavaScript, necesitamos vincular un archivo con la extensión .js al final de nuestro HTML. 

  <script src="comportamiento.js"></script>
</body>

En JavaScript (y en programación) podemos encontrar:

  • Valores: literales, numéricos, booleanos
  • Variables
  • Funciones
  • Loops
  • Eventos

Las variables son "comodines" o identificadores que pueden tener un valor u otro. Nos sirven para manipular datos que no conocemos.

Las funciones nos permiten crear código reutilizable, que solo se activará cuando lo llamemos.

Las loops nos permiten crear tareas de forma repetitiva o cíclica.

Los eventos nos permiten asociar funciones a sucesos concretos (hacer click, cargar la página, etc).

jQuery es un conjunto de funciones prefabricadas para que nos sea más fácil escribir y aprender JavaScript.

Es como aprender a andar en bicicleta con ruedecitas.

Para usar jQuery, tenemos que descargarlo de jQuery.com y cargarlo antes que nuestro archivo de JS. 

  <script src="jquery.js"></script>
  <script src="comportamiento.js"></script>
</body>
