HTML Syntax

<tag>

HTML tags are marked with < > angle brackets.

<tag> </tag>

Most HTML elements have an opening and a closing tag. 

<tag>
  Content
</tag>

The content between the opening and closing tags is associated to that tag.

<p>

The <p> tag groups each paragraph of a text. 

<p>
  This is an example.
</p>

<br>

The <br> tag makes a text line break.

<br> </br>

HTML elements that cannot have content do not need a closing tag.

<br />

Sometimes, these types of elements can be written this way, with the symbol / at the end of the tag. 

<p class="example">
  This is an example.
</p>

HTML tags can have attributes, which allow you to apply meta information for the browser. They are always written in the opening tag. 

<p class="example" id="main">
  This is an example.
</p>

Each attribute is associated to its value with the = symbol and the value or values are quoted. 

<p class="example alternative" id="main">
  This is an example.
</p>

The class attribute can have more than one value, separated by spaces (the classes are combined). The id attribute can only have one value, and there cannot be two id with the same value on the same page. 

<!-- This is a comment -->

HTML comments allow us to organize our code and tell the browser to ignore certain blocks.
