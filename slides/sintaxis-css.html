Sintaxis CSS

<link rel="stylesheet" href="estilos.css" >

Vincularemos una hoja de estilos a un archivo HTML a través de la etiqueta <link>. 

selector {
  propiedad: valor;
}

Escribimos las reglas de CSS siguiendo el esquema selector { propiedad: valor; }. Cada instrucción se finaliza con un punto y coma. Todas las instrucciones englobadas entre llaves afectan a dicho selector. 

h1, h2, h3 {
  font-size: 24px;
}

Podemos agrupar selectores separándolos por comas para asignarles los mismos estilos. 

h1 {font-size: 24px;}
h1 {font-size: 20px;}

En caso de dos declaraciones que entran en conflicto (mismo selector, misma propiedad), la declaración posterior prevalece.
