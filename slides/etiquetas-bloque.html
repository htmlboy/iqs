Etiquetas de bloque

<em> <strong> <span> <a> <img>

Estas etiquetas son de línea (no provocan un salto del contenido a continuación). 

<p> <h1> <h2> <h3> <h4> …

Estas etiquetas son de bloque (provocan un salto de línea), pero están preparadas para contener textos (no otros bloques). 

<div>

Esta etiqueta de bloque sirve para crear estructuras, bloques y agrupaciones de bloques.

Los <div> pueden agruparse unos dentro de otros para crear estructuras más complejas.

Los <ul> nos permiten crear listas. Cada elemento de una lista va dentro de un <li>.

Los <ol> nos permiten crear listas numeradas. El resto de las reglas son iguales que con los <ul>.
