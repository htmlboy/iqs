Webfonts

When setting a typeface family, especially if we trust that it would be installed on the user's computer, we can declare alternatives, separated by commas.

If we load a typeface from an external service, such as Google Fonts, we will have to load their stylesheet, with the appropriate weights, and follow any other instructions. 

@font-face {
  font-family: Lobster;
  src: url(fonts/lobster-bold.woff) format("woff");
  font-weight: bold;
  font-style: normal;
}

If we load webfonts from our web archives, we will put an @font-face declaration at the beginning of our CSS for each file of each font weight, indicating its family, weight and style. 

@font-face {
  font-family: Lobster;
    src: url(fonts/lobster-bold.woff2) format("woff2"),
         url(fonts/lobster-bold.woff) format("woff"),
         url(fonts/lobster-bold.ttf) format("truetype");
  font-weight: bold;
  font-style: normal;
}

We can set several file formats (woff and woff2 are the standard, and woff2 the lightest). The browser will download the first file that is capable of interpreting.
