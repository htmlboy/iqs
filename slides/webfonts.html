Webfonts

Cuando declaramos una familia tipográfica, especialmente si confiamos en que esté instalada en el ordenador del usuario, podemos declarar alternativas, separadas por comas.

Si cargamos una tipografía desde un servicio externo, como Google Fonts, debemos asegurarnos de cargar su hoja de estilos, con los pesos adecuados, y cualquier otra instrucción. 

@font-face {
  font-family: Lobster;
  src: url(fonts/lobster-bold.woff) format("woff");
  font-weight: bold;
  font-style: normal;
}

Si cargamos webfonts desde los archivos de nuestra web, pondremos una declaración @font-face al principio de nuestra CSS para cada archivo de cada peso tipográfico, indicando la familia, peso y estilo del mismo. 

@font-face {
  font-family: Lobster;
    src: url(fonts/lobster-bold.woff2) format("woff2"),
         url(fonts/lobster-bold.woff) format("woff"),
         url(fonts/lobster-bold.ttf) format("truetype");
  font-weight: bold;
  font-style: normal;
}

Podemos indicar varios formatos de archivo (woff y woff2 son el estándar, y woff2 el más ligero). El navegador descargará el primer archivo que sea capaz de interpretar.
