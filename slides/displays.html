Displays

Los elementos HTML pueden ser de bloque (provocan un salto de línea) o de línea (se alinean como texto).

Escribiendo en nuestra CSS display:inline podemos hacer que un elemento de bloque se muestre como un elemento de línea.

Y escribiendo display:block podemos hacer que un elemento de línea se muestre como un elemento de bloque.

Los elementos de línea no aplican margins y paddings de la manera esperada (desplazando contenido superior o inferior). Si queremos este funcionamiento sin provocar un salto de línea, podemos utilizar el intermedio display:inline-block.
