Bordes

.example {
  border: 5px solid blue;
}

Para asignar un borde debemos declarar la anchura y el tipo. También podemos asignar el color, si no por defecto es el color del cuerpo de texto. 

.example {
  border-width: 5px;
  border-type: solid;
  border-color: blue;
}

La propiedad border se puede desglosar en 3 propiedades border-width, border-type, border-color si preferimos modificar solo una de ellas.

También podemos hacer referencia al borde superior, derecho, inferior o izquierdo.
