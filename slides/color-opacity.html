Color and opacity

.example{
  color: red;
  color: #ff0000;
  color: rgb(255, 0, 0);
}

There are several ways to set a color: keyword, hexadecimal, rgb()â€¦

We can manipulate opacity or transparency through the opacity property or the rgba() alpha channel. 

.example{
  color: hsl(302, 100%, 50%);
}

The hsl() system is another way of declaring color that can be more intuitive. 

.example{
  color: hsla(302, 100%, 50%, 0.5);
}

The hsla() system works the same as hsl(), but with the alpha channel.

You can try an online color editor at hslpicker.com.
