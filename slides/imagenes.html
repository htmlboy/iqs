Imágenes

<img>

La etiqueta <img> se utiliza para mostrar imágenes de contenido en una web. 

<img src="gato.jpg">

El atributo src sirve para indicar la ruta para llegar al archivo de la imagen, incluyendo el nombre del propio archivo. 

<img src="gato.jpg" alt="Un gato pardo">

Si la imagen tiene valor informativo, debemos poner un texto que la describa en el atributo alt y así podrá ser interpretada por lectores de pantalla. 

<img src="imagen.jpg" alt="">

Si la imagen no tiene valor informativo, debemos dejar el atributo alt vacío. 

<img src="images/gato.jpg" alt="">

Si una imagen se encuentra dentro de una carpeta respecto a la página (archivo HTML) que la llama, se debe poner el nombre de la carpeta.

Si la carpeta de imágenes se encuentra en un nivel de carpetas superior respecto al archivo HTML, usaremos dos puntos (..) para subir un nivel de carpetas. 

<img src="../images/gato.jpg" alt="">

<img src="../../images/gato.jpg" alt="">

Para cada nivel de carpetas que queramos subir, añadiremos dos puntos más.
