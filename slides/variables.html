Variables

Y organización de archivos

html {
  --color: hotpink;
}
p {
  color: var(--color);
}

CSS nos permite centralizar valores (tamaños, colores…) en variables, para facilitar los cambios de manera consistente. Las variables se pueden declarar de forma global en el selector html o :root (el nombre de las variables siempre empieza por --) y emplearse con la función var() y el nombre de la variable.

Podemos declarar las variables en otro contexto que no sea el :root, y estarán disponibles en este contexto. También podemos sobreescribirlas en un nodo concreto. 

@import url(general.css);
@import url(tipografia.css);

El archivo principal de estilos puede incorporar otros archivos externos con la regla @import. Esto permite tener un solo archivo de estilos vinculado en el head, pero que la propia CSS esté dividida en varios archivos para facilitar el trabajo. 

@import url(variables.css);
@import url(general.css);
@import url(tipografia.css);
@import url(botones.css);
@import url(layout.css);
@import url(cabecera.css);
@import url(footer.css);
@import url(galeria.css);
@import url(video.css);
@import url(productos-relacionados.css);

Divide la CSS en diferentes archivos según su función. Ordénalos de lo más general a lo más específico. Centraliza las variables en uno o varios archivos para tener estilos más consistentes.
