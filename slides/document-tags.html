Document tags

<html> </html>

The <html> tag wraps almost everything: the <head> and <body> tags. 

<html>
  …
  <body>
    <h1>Content</h1>
  </body>
</html>

The <body> tag includes all the content of the page (texts, images, etc). 

<html>
  <head>
    …
  </head>
  <body>
    …
  </body>
</html>

The <head> tag wraps the meta information that the browser needs to know in order to render the page: stylesheets, <title>, encoding, etc. 

<head>
  <title> Page title </title>
</head>

The <title> tag allows you to give the page a title (different from a heading), that will appear in the browser tab. 

<head>
  <meta charset="utf-8">
</head>

The <meta> tag with the charset attribute allows us to tell the browser which encoding is using the page, so that the special characters of our text (ñ, ç, ^, `, ´, etc.) are correctly interpreted. 

<head>
  <link rel="stylesheet" href="styles.css">
</head>

The <link> tag with the attributes rel="stylesheet" and href allows us to load a stylesheet. 

<!DOCTYPE html>

The <DOCTYPE> tag is used to indicate that a web follows the web standards, and is not created in pre-standard times. It is placed at the beginning of everything, outside the <html> tag. 

<!DOCTYPE html>
<html>
  <head>
    <title> … </title>
    <meta charset="utf-8">
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    …
  </body>
</html>

This is the complete structure of the document tags of a web page.
