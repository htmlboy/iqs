Estilos para listas

ul {
  list-style: disc;
  list-style-position: outside;
  padding-left: 40px;
}

Los listados (ul y ol) tienen un conjunto de estilos por defecto, que probablemente querremos sobrescribir.

Podemos modificar estos estilos con propiedades como list-style-type o list-style-position.
