Introduction to the WWW

What is the Internet?

It's the global system of interconnected computer networks that use the Internet protocol suite (TCP/IP) to link devices worldwide.

How old is the Internet?

Its origins date back to 1969, when the first computer connection, known as ARPANET, was established between three universities in California.

What is the WWW?

It is a system of distribution of hypertext or hypermedia documents interconnected and accessible through the Internet.

How old is the WWW?

The Web was developed between March 1989 and December 1990, by the Englishman Tim Berners-Lee with the help of Belgian Robert Cailliau while working at CERN in Geneva, Switzerland.

What happens when accessing a website?

To publish a website we need to host the necessary files on a computer permanently connected to the Internet, with specific software and a physical address, an IP (example: 79.153.70.215).

We will hire this service, known as hosting, to specialized companies, paying an annual rent.

In addition, we need to hire a domain (also annual rent). The Internet domains are centralized in a registry controlled by the non-profit association ICANN, and allow you to associate a name and extension (www.example.com, .net, .org â€¦) with the IP of a server.

When we enter an address in the browser (or follow a link), the browser checks the IP associated with the domain and contacts the server.

Through the rest of the address, the server software knows what HTML file to return. When we access the root of the domain (www.example.com), the server usually shows the index.html file, the home page.

The browser downloads a text file (HTML), which has information about its contents, but also information about the dependent files (images, stylesheets and JavaScript scripts).

Once it has downloaded images and stylesheets (CSS), it has enough information to display the page in the format we expected.

And once it has downloaded and interpreted the JavaScript, it can add advanced interaction to the page (which initially does not differ much from a Word document).

How has the WWW evolved?

The WWW was developed between 1989 and 1990, and was published in 1992.

Websites with basic styles were developed using HTML styling tags and very limited interaction.

Space Jam website, published in 1996.

In 1995 the first version of JavaScript was launched, which would allow you to add programming instructions that are interpreted directly in the browser, and thus achieve much more interactive websites.

In 1996, the first version of CSS, or Cascading Style Sheets , was launched, the styling language that would allow separating the content (HTML) from the visual presentation.

For several years, the web was the property of the desktop computer, whose monitors were gaining size and resolution, but they were quite uniform.

In 2007 the iPhone was released, the first smartphone, and suddenly we were forced to create a website that can also work on these devices (and those that will come).

In 2010 Ethan Marcotte conceived the Responsive Web Design, a set of techniques and rules for developing websites that adapt to different devices.

Between 2010 and 2014, frameworks such as Angular, React or Vue were created, based on JavaScript and designed to create web apps: websites with high interaction in the browser without reloading the page.

The popularization of these libraries and the complexity of the developments and tools changed the paradigm of how we view web sites.

We no longer talk about pages, but components.

We identify the elements that compose and repeat throughout an interface, we isolate them and centralize their design and development.
