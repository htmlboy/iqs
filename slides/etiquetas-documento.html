Etiquetas de documento

<html> </html>

La etiqueta <html> engloba casi todo: las etiquetas <head> y <body>. 

<html>
  …
  <body>
    <h1>Contenido</h1>
  </body>
</html>

La etiqueta <body> engloba todo el contenido de la página (textos, imágenes, etc). 

<html>
  <head>
    …
  </head>
  <body>
    …
  </body>
</html>

La etiqueta <head> engloba la meta información que necesita el navegador para renderizar la página: hojas de estilo, <title>, codificación, etc. 

<head>
  <title> Título de la página </title>
</head>

La etiqueta <title> permite dar un título a la página (diferente de un encabezado) y aparecerá en la pestaña del navegador. 

<head>
  <meta charset="utf-8">
</head>

La etiqueta <meta> con el atributo charset nos permite indicar al navegador en qué codificación están los caracteres especiales de nuestro texto (ñ, ç, tildes, etc), para que no se confundan con otros. 

<head>
  <link rel="stylesheet" href="estilos.css">
</head>

La etiqueta <link> con los atributos rel="stylesheet" y href permite cargar una hoja de estilos. 

<!DOCTYPE html>

La etiqueta <DOCTYPE> se emplea para indicar que una web sigue los estándares web, y no está creada en la época pre-estándares. Se coloca al principio de todo, fuera de la etiqueta <html>. 

<!DOCTYPE html>
<html>
  <head>
    <title> … </title>
    <meta charset="utf-8">
    <link rel="stylesheet" href="estilos.css">
  </head>
  <body>
    …
  </body>
</html>

Esta es la estructura completa de las etiquetas de documento.
