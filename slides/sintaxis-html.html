Sintaxis HTML

<etiqueta>

Las etiquetas HTML se marcan con los símbolos < >

<etiqueta> </etiqueta>

La mayoría de etiquetas tienen apertura y cierre. 

<etiqueta>
  Ejemplo de contenido
</etiqueta>

El contenido entre la etiqueta de apertura y la de cierre está asociado a esa etiqueta.

<p>

La etiqueta <p> sirve para agrupar cada párrafo de un texto. 

<p>
  Esto es un ejemplo.
</p>

<br>

La etiqueta <br> sirve para hacer un salto de línea en un texto.

<br> </br>

Las etiquetas que no pueden tener contenido no necesitan etiqueta de cierre.

<br />

En ocasiones, este tipo de etiquetas se pueden escribir de esta forma, con el símbolo / al final de la misma. 

<p class="ejemplo">
  Esto es un ejemplo.
</p>

Las etiquetas HTML pueden tener atributos, que permiten aplicar metainformación que el navegador podrá interpretar. Se ponen siempre en la etiqueta de apertura. 

<p class="ejemplo" id="principal">
  Esto es un ejemplo.
</p>

Cada atributo se asocia a su valor con el símbolo = y el valor o valores se entrecomillan. 

<p class="ejemplo alternativo" id="principal">
  Esto es un ejemplo.
</p>

El atributo class puede tener más de un valor, separados por espacios (las clases se combinan). El atributo id solo puede tener un valor, y no puede haber dos id con el mismo valor en una misma página. 

<!-- Esto es un comentario -->

Los comentarios de HTML permiten organizar nuestro código y decirle al navegador que ignore ciertos bloques.
