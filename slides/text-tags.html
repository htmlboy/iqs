Text tags

<p>

The <p> tag groups each paragraph of a text.

<br>

The <br> tag makes a text line break inside a paragraph or a text line.

<h1>

Identifies the main heading of a page. Normally there is only one per page. 

<h1>
  First level heading
</h1>

<h1>
<h2>
<h3>
<h4>
<h5>
<h6>

Each of these headings refer to their importance within the text, never to the order in which they appear.

<em>

Adds emphasis on a particular piece of text. It is usually rendered as italic. 

<p>
  We can emphasize specific words.
</p>

<strong>

Indicates that one or more words have importance within the text. It is usually rendered as bold. 

<p>
  Or we can give importance
  to specific words.
</p>

<span>

It groups a piece of text and allows us to assign a class. 

<p>
  The span tag
  lets us to select a string of text.
</p>
