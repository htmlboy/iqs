Estilos para enlaces

a {
  color: blue;
  text-decoration: none;
}

Los enlaces tienen color y text-decoration por defecto, por lo que hay que sobrescribirlos explícitamente. 

a:hover {
  text-decoration: underline;
}

Las pseudo-clases (o "estado") se escriben con dos puntos (:). El estado de :hover indica cuando el ratón pasa sobre el enlace. 

a:active {
  color: orange;
}

El estado de :active indica el momento en el que se presiona el enlace. 

a:visited {
  color: blue;
}

El estado de :visited indica que el enlace ha sido visitado. En muchos casos querremos sobrescribir este estado para que mantenga el color original. 

a:focus {
  background-color: lightgrey;
}

El estado de :focus indica que el foco del navegador está en ese elemento en particular. Interesante especialmente si navegamos con el teclado.
