Media queries

y layout responsive

La base de una web responsive es una estructura en anchuras porcentuales, capaz de adaptarse a la anchura del navegador.

Añdiendo a nuestra CSS img {max-width: 100%;} nos aseguramos de que las imágenes se reduzcan cuando su contenedor es más pequeño que su tamaño natural. 

@media (max-width: 700px) {
  .example {background-color: red;}
}

Una media query es una regla condicional (la condición se pone entre paréntesis y hace referencia a las condicidiones de la ventana del navegador o del dispositivo).

El código CSS que hay entre las llaves de una media query solo se aplicará si la condición se cumple.

Una media query de max-width solo se aplicará cuando la ventana sea igual o menor a la anchura que hayamos marcado.

Una media query de min-width solo se aplicará cuando la ventana sea igual o mayor que la anchura que hayamos marcado.

Podemos acotar una media query entre un mínimo y un máximo con la palabra and y una segunda condición.
