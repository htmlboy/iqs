Anchor styles

a {
  color: blue;
  text-decoration: none;
}

Anchor elements have color and text-decoration by default, so you have to explicitly overwrite them. 

a:hover {
  text-decoration: underline;
}

Pseudo-classes (or states) are written with a colon (:). The :hover state indicates when the mouse passes over the anchor. 

a:active {
  color: orange;
}

The :active state indicates the moment the anchor is pressed. 

a:visited {
  color: blue;
}

The :visited state indicates that the link has been visited. In many cases this state will be overwritten to keep the original color. 

a:focus {
  background-color: lightgrey;
}

The :focus state indicates that the focus of the browser is on that particular element. It's especially interesting when browsing with the keyboard.
