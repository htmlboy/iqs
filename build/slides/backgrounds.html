Backgrounds

.example {
  background-color: #dd6a92;
}

The background color of an HTML element is declared with background-color. 

.example {
  background-image: url(path/image.jpg);
}

The background-image property sets an image as a background. 

.example {
  background-image: url(path/image1.jpg),
                    url(path/image2.jpg);
}

We could set several background images, separated by commas.

The background-repeat property defines if the background image is repeated and how. The background-position property allows us to align the image across the background. 

.example {
  background: hotpink url(image.jpg) no-repeat right top;
}

The background property allows us to assign the values of several background properties together, separated by spaces.

The background-size property controls the size at which the background images appear.

With the background-image property we can also set a gradient. In the linear-gradient() or radial-gradient() functions we will set the angle or position and color stops.
