Position

We can take out an element of its natural flow within a page with the property position: absolute.

An absolutely positioned element appears in its natural place by default, but without displacing other contents (and sometimes overlapping them).

The position of this element can be controlled with top, bottom, left or right properties.

To absolutely position an element inside another element (instead of the browser's window), the parent element needs to be set to position: relative .

In the case of several positioned elements, the order in which they're painted can be controlled with the z-index property.

As an alternative to position: absolute, we can use position: fixed, which keeps a fixed element within the browser window.

And position: sticky, which turns a regular element into a fixed element once the user has scrolled to some extent.
