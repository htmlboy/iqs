Displays

HTML elements can be block-type (cause a line break) or line-type (they align as text).

By writing display:inline we can make a block element behave as a line element.

And by writing display:block we can make a line element behave as a block element.

Line elements do not apply margins and paddings as expected (moving upper or lower content).
If we want this behaviour without causing a line break, we can use the in-between display:inline-block.
