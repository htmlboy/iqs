Media queries

and responsive layouts

The core of a responsive web is a fluid structure, based in percentage widths, capable of adapting to the browser's width.

By adding img {max-width: 100%;} to the CSS we ensure that the images are reduced when their container is smaller than their natural size. 

@media (max-width: 700px) {
  .example {background-color: red;}
}

A media query is a conditional rule (the condition is placed between parentheses and refers to the conditions of the browser window or the device).

The CSS code between the {} of a media query will only be applied if the condition is met.

A max-width media query will only be applied when the window is equal to or smaller than the specified width.

A min-width media query will only be applied when the window is equal to or greater than the specified width.

We can narrow a media query between a minimum and a maximum with the and keyword and a second condition.
