CSS Units

The values in px are absolute. The values in em depend on the font-size inherited, and indicate a proportion to this inherited size. The values in % work similarly to the em when applied to typography (when applied to a width, they depend on the available space).

The values in rem are proportional to the font size of the html tag directly, regardless of inheritance.

The units related to the viewport (vw for width, vh for height) work in a similar way to percentages (100vh = 100% of the screen height).

It is advisable to avoid making something 100vw wide because in some operating systems can cause horizontal scrolling.

The viewport width unit (vw) is interesting to create adapting headers.

In addition to vw and vw there are the units vmin (1% of the screen width or height, depending on which side is smaller) and vmax (1% of the screen width or height, depending on which side is larger).
