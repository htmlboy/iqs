Borders

.example {
  border: 5px solid blue;
}

To properly set a border we must declare the width and type. We can also assign the color, that by default is the color of the body text. 

.example {
  border-width: 5px;
  border-type: solid;
  border-color: blue;
}

The border property can be broken down into 3 properties: border-width, border-type and border-color, in case we want to modify only one of them.

We can also refer to the top, right, bottom or left edge.
