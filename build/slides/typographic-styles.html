Typographic styles

Typographic styles are inherited. If we establish a typographic family for the body, it affects their descendants.

The typographic family is set with font-family.

the typographic size is set with font-size.

The typographic weight is set with font-weight.

The typographic variant is set with font-style (normal or italic).

The underline is set with text-decoration.

The line height is set with line-height.

The text color is set with the color property (not to be confused with background-color, for the backgrounds).
