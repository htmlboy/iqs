Hyperlinks

<a>

The <a> tag (anchor) is used to create links to other web pages. 

<a href="page.html">Example of text</a>

The href attribute is used to indicate the path to reach the web page we want to link. 

<a href="http://google.com">Example of text</a>

If the page we link is external, we must write the full address, including http://. 

<a href="http://google.com" target="_blank">Example of text</a>

If we want it to load in a new tab, we will use the target="_blank" attribute.

We can link to specific sections of a page that have the attribute id. The id are unique identifiers, and there shouldn't be two with the same value on a page.

We will refer to the id with a hash: # 

<a href="mailto:javier@swwweet.com">Send me an email</a>

We can link to an email with the prefix mailto:, and this will open the email program that you have installed by default on your computer.
