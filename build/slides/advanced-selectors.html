Advanced selectors

The child selector (>) only applies to a direct descendant (rather than nested ones). The + selector refers to the adjacent element below.

The pseudo-classes :first-child and :last-child refer to the position that an element occupies (if it is first or last) in a context (or HTML node).

With the pseudo-class :nth-child() we can apply styles to elements in a specific position. We refer to the element in a position by giving a number (argument) inside the parentheses. If we add an n, we indicate repetition, and adding or subtracting a number will modify the point at which it begins to count.

With :first-of-type, :last-of-type or :nth-of-type() we can refer to the position of an element according to its type of HTML tag.

The [] attribute selector can be applied to an element that has an attribute or which attribute is equal to a certain value.

The :not() selector applies an exception to the rule (a class, an attributeâ€¦). It does not allow combinations of parameters.

The pseudo-elements ::before and ::after allow us to add content at the beginning and end of an HTML tag, providing this tag can have contents (they do not work with <img>, for example).

The pseudo-element ::first-line allows us to apply styles to the first line of a block of text. ::first-letter applies styles to the first letter, and ::selection to the selected text.
