Flexbox

By setting display: flex to an element, it becomes a flexbox space, and its direct descendants (children) will line up on the main axis.

We can change the main axis direction with flex-direction: column. The default value is row, but there is also row-reverse and column-reverse.

A flexbox space will always try to sort all the content on the main axis, even if it has no room for it. We can modify this behavior with the flex-wrap: wrap property (it's nowrap by default).

The default width of an element within a flexbox space can be controlled with flex-basis. We can also control how much of the remaining space should be asigned to an item with flex-grow (0 by default). And we can control the shrink factor of an item when there is no available space with flex-shrink (1 by default).

We can change the order of the elements within a flexbox space. All elements have an order: 0 by default, but they will always be arranged from lower to higher values.

We can align the content on the main axis with the justify-content property.

We can align the content on the cross axis with the align-items property.

We can change the alignment of a particular element in the cross axis with align-self. We can also tweak the main axis if we use margin-top / bottom / left / right: auto.
