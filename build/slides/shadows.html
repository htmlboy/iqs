Shadows

A box shadow is normally declared with various values separated by spaces: displacement on the X axis, on the Y axis, blur and color.

If we add one more value after the blur, it will be interpreted as the spread or “shadow padding”, a color spacing before the blurring.

Preceding the shadow values with the inset parameter, the shadow will be rendered inside the box.

We can assign as many shadows as we want, separated by commas.

The text-shadow property allows shadows to be applied to the text the same way as with box-shadow. The main difference is that it does not accept the inset value.
